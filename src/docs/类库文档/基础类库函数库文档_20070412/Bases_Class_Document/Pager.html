不依赖于任何实际数据环境的分页类

注意：本分页类不会进行任何数据级别的操作， 忽略数据是来自数据库、文件、数组、Socket网络数据、Web Service等等获取的数据， 本数据类都不关心，只是单纯的关心我们的分页显示结果，具体数据操作需要自行在调用程序中描述
